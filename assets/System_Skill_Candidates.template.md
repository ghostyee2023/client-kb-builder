 # 系统级 Skill 候选清单
 
 > 本清单列出建议安装的系统级 skill。系统级 skill 是通用能力（文档、表格、搜索、平台集成等），不等同于你业务专属的项目级 skill。
 > 
 > 按需安装，安装后登记到能力开关表。未经验证的 skill 默认状态为 `manual`，跑通后再改 `on`。

 ## 安装说明
 
 1. 先装通用基础（documents、spreadsheets）。
 2. 再按你的业务类型加装行业推荐。
 3. 每个 skill 安装后，到能力开关表登记，默认状态 `manual`。
 4. 跑通首条业务流后，把验证过的 skill 从 `manual` 改为 `on`。
 5. 不确定要不要的不装，跑几轮业务流后按实际卡点补。
