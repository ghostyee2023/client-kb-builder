# 律师客户知识库

这是一个从空白根目录搭建的律师业务知识库样例。

首版只跑通一条核心工作流：

```text
客户咨询
-> 冲突检查
-> 风险初筛
-> 案情访谈
-> 证据目录
-> 代理方案
-> 办案交付
-> 单案复盘
-> 模板与案例沉淀
```

## 首版原则

- 先服务律师真实办案动作，不做全所管理系统。
- 先把咨询到交付跑通，再扩展合同审查、法律顾问、诉讼证据管理等工作流。
- 客户身份、案件细节、证据材料默认保密，不进入公共案例资产。
- AI 只能辅助整理、摘要、生成待确认草稿，不能替代律师专业判断。

- AI 执行规则：`AGENTS.md` / `CLAUDE.md`
- 今日待处理：`00_律所工作台/今日待处理.md`
- 案件流程导航：`00_律所工作台/案件流程导航.md`
- 新咨询记录：`02_客户咨询收件箱/新咨询记录卡.md`

- 单案复盘：`09_律师复盘/01_单案复盘/单案复盘模板.md`
