# 案件流程导航

## lawyer_case_delivery_flow

```text
新咨询
-> 记录客户与对方主体
-> 冲突检查
-> 风险初筛
-> 案情访谈
-> 证据目录
-> 代理方案与报价
-> 委托签署
-> 办案交付
-> 结案归档
-> 单案复盘
-> 模板升级
```
